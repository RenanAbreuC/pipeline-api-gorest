# Projeto ETL Simples com API GoRest

Este projeto extrai, processa e salva dados usando a API pública GoRest, a biblioteca Requests e o Pandas. Serve para prática de conceitos de Engenharia de Dados, autenticação, métodos HTTP e manipulação de DataFrames.

## Estrutura

- `src/main.py`: Script principal
- `data/`: Onde ficam os dados gerados (CSV)
- `.env`: Armazena o token de autenticação (não vai para o GitHub!)
